# Advanced Tic-Tac-Toe System

A robust, enterprise-grade Tic-Tac-Toe system built in C# utilizing core Object-Oriented Programming (OOP) principles. The project features advanced game modes, intelligent AI decision-making, and persistent data storage.

---

## 🎯 Project Architecture & Evolution
To demonstrate scalable software architecture, this project follows a **Monorepo approach** tracking the evolution from a classic CLI application to a modern desktop GUI:

*   📂 **`Console-Version/`**: A fully functional, production-ready console application featuring clean OOP abstraction, structured data models, and custom exceptions.
*   📂 **`WinForms-Version/`** *(Upcoming)*: Transitioning the core game logic into a modern Windows Forms (GUI) desktop application.

---

## 🚀 Key Features

*   **Diverse Game Modes:** Supports Player vs Player (Local) and Player vs AI.
*   **Intelligent AI Component:** Engineered logic for automated opponent moves.
*   **Tournament & Leaderboards:** Features tournament capabilities with state persistence.
*   **Data Persistence:** Uses C# **File Handling** to read/write match histories, user statistics, and persistent leaderboards.
*   **Robust Architecture:** Fully decoupled components using Interfaces, Custom Enums, and dedicated Utility classes.
*   **Exception Handling:** Integrated custom validation and try-catch blocks to ensure crash-proof runtime operations.

---

## 🛠️ Technical Stack & Concepts Used

*   **Language:** C# (.NET Core / .NET Framework)
*   **Programming Paradigms:** Object-Oriented Programming (Encapsulation, Abstraction, Polymorphism, Interfaces)
*   **File I/O:** Persistent Text/Data Logging for Tournament Histories
*   **Version Control:** Git & GitHub for clean collaboration and deployment
